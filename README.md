# BookingBot

Professional AI-powered Telegram Booking Bot built with Python and aiogram.

## Features

- AI booking assistant
- Calendar management
- Time slot reservation
- Admin panel
- User profiles
- Notifications
- SQLite database
- OpenAI integration
- Clean architecture

## Tech Stack

- Python
- aiogram 3
- SQLite
- SQLAlchemy
- OpenAI API
- aiohttp
- Loguru
